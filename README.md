# Scholarship Process Improvement

## Project Overview
This project analyzes scholarship communication and response issues in an educational institution where students face delays, unclear information, and repeated visits to the office. The goal is to improve communication, reduce waiting time, and create a structured response process using Business Analysis techniques.

## Problem Statement
Students experience delayed and unclear scholarship updates, resulting in long queues, repeated office visits, miscommunication with academic staff, and unnecessary time loss.

## Objectives
- Identify gaps in the scholarship communication process
- Understand stakeholder roles and responsibilities
- Analyze root causes of delays and miscommunication
- Recommend process improvements to reduce student waiting time
- Improve coordination between academic and administrative teams

## Stakeholders
- Students
- Class Advisors
- Heads of Department (HODs)
- Office Administrators
- Office Staff

## Analysis Approach
- Stakeholder Identification and Analysis
- Root Cause Analysis (5 Whys)
- AS-IS and TO-BE Process Analysis
- RASCI Matrix
- Documentation and Research

## Key Recommendations
- Provide accurate and timely scholarship updates
- Establish a clear communication channel between office staff and academic staff
- Share official updates only through verified channels
- Allocate staff based on query volume
- Create separate handling paths for different scholarship-related queries
- Reduce crowding by structured scheduling or digital updates
